## 1. ブランド構造

### D/ONE（ディーワン）
- **ポジション**: アパレルブランド × ドキュメンタリー型メディア × 8タイプ診断 × 若年層支援
- **思想**: 「One day を、Day One に。」 — 「いつかやりたい」を「今日が最初の一歩」に変える
- **企画の結論**: デザイン性のあるリアルなアパレルブランドを事業の軸に置き、その周辺に診断・支援・メディアを展開する
- **ブランドとしての約束**: 「情報の中から自分らしい始め方を見つけるブランド」「自分らしい始め方を見つけるブランド」
- **4つの柱**:
  1. 着るのは自己表現
  2. 着るのは挑戦の物語
  3. 近くのはタイプは違っても（多様性）
  4. （※4つ目はPDFで確認しきれず）
- **社内向けの最終整理**: 服を売るブランドであると同時に、温かくわかりやすく、人を理解し共に考えるブランド

### KAIWAI（界隈）
- **ポジション**: 学生支援プラットフォーム
- **ロゴ**: 抽象的な曲線シンボル（シルバー/グレー × オレンジ/レッドのグラデーション）
- **バリエーション**: ダーク版（黒背景・白テキスト）/ ライト版（白背景・黒テキスト）
- **ブランドカラー**: オレンジ/レッド + シルバー/グレー + 黒/白

### D/ONEとKAIWAIの関係（要確認）
- D/ONEはアパレルを主軸としたブランド全体像
- KAIWAIは学生支援プラットフォームとしてのサービス名
- 両者の統合度・棲み分けは未確定

## 2. 事業概要

### サービスのコンセプト
「何かやりたいけど、何をすればいいかわからない」学生に対し、診断・分類から段階的支援、実務参加、成果の可視化までを一気通貫で提供するプラットフォーム。

支援を受ける側が支援する側にもなれる「循環構造」を持つことが最大の特徴。

### 事業の本質（会話で整理済み）
- 単なる学生支援ではなく、「学生の意欲を整理し、ガクチカ形成を中心に実務参加・実績化・機会接続まで行う仕組み」
- 入口は広く取りつつ、主軸はガクチカ支援に置く
- 学生から強く課金するモデルではない
- 企業・エージェント連携と制作受託で収益を立てる
- 将来的にはVC/インキュベーション的な役割を目指す

### 全体の流れ
```
SNS集客 → アンケート診断 → ラベリング → 松竹梅振り分け → 支援提供 → 実務参加 → 成果可視化
```

### D/ONE計画での基本フロー
```
認知（SNS発信）→ 診断（自分ごと化）→ 月1企画で動く
```

### D/ONE計画でのマーケティングフロー
```
SNS発信 → 診断 → 参加企画 → ブランド価値認定 → アパレル購入
```

### 3つの機能軸

| 機能軸 | 内容 | 役割 |
|--------|------|------|
| ラベリング・マッチング | アンケートによる志向・関心・現在地の分類 | 支援・役割分担の基盤となる入口 |
| サービス提供 | Web制作・コンテンツ・メディア・発信支援 | 「思い」や「アイデア」を形にするフェーズ |
| 逆算タスク提供 | 目標達成に必要なタスクを細分化して提示 | 曖昧な人にも具体的な行動を与える |

## 3. 松竹梅コース設計

### 松（入口・参加層）

| 項目 | 内容 |
|------|------|
| 対象 | 何をしたいか明確でない人 / 最初の行動がわからない人 / 実績を積みたい人 |
| 提供内容 | 診断による自己理解 / 軽いマッチング / 初歩的な逆算タスク / 認証制実務タスク参加 |
| 得られるもの | チームへの参加感 / ガクチカ・実績づくり / 小さな成功体験 / 誰かの挑戦を支える感覚 |
| 位置づけ | 単なる初心者層ではなく、竹・梅へ進むための育成・参加層 |

### 竹（実行・具体化層）

| 項目 | 内容 |
|------|------|
| 対象 | 方向性が決まっている人 / 動き始めている人 / 外部支援があると前進しやすい人 |
| 提供内容 | 制作支援 / コンテンツ支援 / 発信支援 / 目標達成のための具体的サポート |
| 位置づけ | 「やりたいこと」を実際に動かす実行フェーズ |

### 梅（伴走・選抜層）

| 項目 | 内容 |
|------|------|
| 対象 | 有望と判断された個人・チーム / メディア化・ストーリー化に耐えうる人 |
| 提供内容 | 伴走型支援 / 実務面支援 / ブランド化・メディア化 / 外部発信協力 |
| 重要な役割 | 梅の実務を細分化し、松層に開放することでプラットフォーム内に循環を生む |

## 4. ラベリング設計

### 最上位カテゴリ（5つ）

| カテゴリ | 意味 | 内包する活動 |
|---------|------|------------|
| **ガクチカ** | 学生時代に力を入れた経験として就活・自己PRに話せる経験 | SNS運用 / Web制作 / コンテンツ制作 / イベント運営など |
| **留学** | 海外経験・異文化交流・語学力に関わる経験 | 留学準備 / 語学学習 / 英語カフェ |
| **資格** | 検定・資格取得によるわかりやすいスキル証明 | 人気資格 / 学習計画 / 検定挑戦 |
| **サークル** | 団体活動・チーム活動・所属感 | 文化系 / 体育会 / スポーツ / 運営 |
| **その他** | まだ方向性が明確でない人の入口カテゴリ | 追加アンケートで4カテゴリへ誘導 |

### 設計思想
- 「その他」は放置しない。最終分岐ではなく、追加アンケートによって必ず4カテゴリのいずれかに接続する
- 「何に興味ありますか？」と直接聞かず、間接的な質問で志向を引き出す
- 目的: まだ曖昧な状態の人を、具体的な行動導線へ近づけるための誘導設計

### 誘導の3ステップ

| ステップ | 内容 |
|---------|------|
| 第一段階 | ユーザーは最初に5カテゴリに触れる。「その他」を含む導入 |
| 第二段階 | 「その他」を選んだ人に追加アンケート（5問）で志向を探る |
| 第三段階 | 回答をもとに「ガクチカ型」「留学型」「資格型」「サークル型」のいずれかに誘導 |

### 「その他」細分化用 5問アンケート

#### Q1. 今いちばん得たいものに近いのはどれですか？

| 選択肢 | 回答内容 | 対応ラベル |
|--------|---------|----------|
| A | 就活や自己PRで話せる経験 | ガクチカ |
| B | 語学力や海外に関わる経験 | 留学 |
| C | わかりやすいスキルや証明 | 資格 |
| D | 仲間と一緒に取り組む居場所や活動 | サークル |

#### Q2. 何か始めるとしたら、いちばん続けやすいのはどれですか？

| 選択肢 | 回答内容 | 対応ラベル |
|--------|---------|----------|
| A | 何かを作る・発信する・運営する | ガクチカ |
| B | 英語や海外に関する準備を進める | 留学 |
| C | 勉強して試験や検定に挑戦する | 資格 |
| D | 誰かと一緒に活動したりイベントに参加する | サークル |

#### Q3. 周りから評価されたい形として近いのはどれですか？

| 選択肢 | 回答内容 | 対応ラベル |
|--------|---------|----------|
| A | 「ちゃんと行動して経験を積んでいる人」 | ガクチカ |
| B | 「新しい環境に挑戦している人」 | 留学 |
| C | 「努力して知識や能力を身につけた人」 | 資格 |
| D | 「チームの中で活躍できる人」 | サークル |

#### Q4. 今の自分に足りないと感じるものはどれですか？

| 選択肢 | 回答内容 | 対応ラベル |
|--------|---------|----------|
| A | 実績や行動経験 | ガクチカ |
| B | 視野の広さや国際感覚 | 留学 |
| C | 専門性やわかりやすい武器 | 資格 |
| D | 人とのつながりや所属感 | サークル |

#### Q5. 最初の一歩として、少しワクワクするのはどれですか？

| 選択肢 | 回答内容 | 対応ラベル |
|--------|---------|----------|
| A | SNS運用、Web制作、企画、発信などに関わる | ガクチカ |
| B | 留学準備、語学学習、海外情報収集を始める | 留学 |
| C | 資格の勉強計画を立てる | 資格 |
| D | コミュニティやサークル活動に参加する | サークル |

### 判定ロジック
- **基本ルール**: 最多回答のカテゴリに振り分け
- **同点の場合**: Q1を最重視。Q1 > Q5 の優先順位で判定
- **おすすめ表示文**: 各タイプに応じた説明文を表示

### おすすめ表示文の例

| タイプ | 表示文 |
|--------|--------|
| **ガクチカ型** | まずは「経験を作ること」が同じにつながりやすいタイプです。SNS運用、Web制作、企画発信、情報発信など、自分で実績の形を作れる活動が向いています。 |
| **留学型** | 海外や異文化にまつわる経験が、あなたの行動のモチベーションになりやすいタイプです。語学学習、留学準備プログラムなどから始めてみましょう。 |
| **資格型** | 目標に挑み、努力した成果がわかりやすい形で残ることにやりがいを感じるタイプです。人気資格の情報収集から始めてみましょう。 |
| **サークル型** | 人とのつながりの中で力を発揮するタイプです。コミュニティ活動やサークル運営に参加してみましょう。 |

## 5. D/ONE 8タイプ診断（企画書より）

D/ONE企画ではラベリングとは別に「8タイプ診断」が言及されている。

- 行動・価値観をもとに分類する8つの性格タイプ
- ブランドとの接点を「自分ごと」にするための仕組み
- 具体的な8タイプの名称・定義はPDFから完全には読み取れず（要追加確認）

### 8タイプとの連動例（D/ONE計画書より）
- 設定: 1〜8タイプに分類
- 交流会: タイプを超えたつながり
- 認証: 3ステップの認証制度

## 7. カテゴリ別支援内容

### ガクチカ
- **インターン**: ヒアリング → 審査 → プロジェクト参加（簡易・本格）
- **起業**: ヒアリング → 審査 → プロジェクト支援
- **人材会社連携**: フォーマット作成 → 探索・開拓 → ベンチャー開拓

### 留学
- **事前準備**: 英語 / TOEIC対策 → 英語カフェ
- **国決め**: 期間・予算設定 → エージェント選定
- **英語カフェ**: コミュニティ運営 → 留学準備と並行

### 資格
- 人気資格一覧の提供
- アンケートによる興味・適性の分類
- 学習計画の作成支援

### サークル
- **入りたい人**: 文化系 / 体育会のマッチング
- **既に入っている人**: 宣伝・集客支援
- **スポーツ**: スポーツ一覧 → 日程（常時）→ SNS集客

## 12. 外部からの客観評価（会話内フィードバック）

### 強み
- 課題設定がリアル（「何かやりたいけどわからない」は大きな需要）
- 松竹梅の段階設計がわかりやすい
- 循環構造（支援される側→支援する側）が核
- 支援内容が抽象論ではなく実務粒度に落ちている
- ガクチカを中核にしつつ周辺ニーズは導線として持つハブ型モデル

### リスク・注意点
- 「なんでも屋」化のリスク
- 無料支援沼（学生向けは価格耐性が低い）
- 品質事故（松層への実務開放による信用毀損リスク）
- 継続率（やりたいことが曖昧な層は離脱もしやすい）
- 循環は自然発生しない（仕事の供給と評価制度が先に必要）
- 「学生を企業に流すだけ」と見られないための学生ファースト設計

### 改善提案（優先順）
1. 対象を絞る（初期はガクチカ × 実務参加）
2. 収益モデルを1本立てる
3. 診断より「次の一歩」の質を高める
4. 評価制度を作る（タスク完了履歴・信頼スコア・実績ポートフォリオ）
